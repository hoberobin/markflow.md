# Code of conduct

We want participation in this project to stay respectful and productive. This document adapts the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/) (v2.1).

## Standards

**Encouraged:** empathy, constructive feedback, accepting responsibility for mistakes, focusing on what helps the project and its users.

**Not acceptable:** harassment, sexualized language or attention, trolling, personal attacks, publishing others’ private information without consent, or other conduct inappropriate in a professional context.

## Scope

This applies in GitHub spaces for this repository (issues, PRs, discussions) and when representing the project in public.

## Reporting

Report concerns to the **maintainers** via [GitHub Issues](https://github.com/hoberobin/markflow/issues). For sensitive matters, use [private security advisories](https://github.com/hoberobin/markflow/security/advisories/new) if appropriate, or contact maintainers through their GitHub profiles.

Maintainers may warn, moderate, or restrict access for repeated or serious violations.
